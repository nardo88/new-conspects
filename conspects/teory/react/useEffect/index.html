Хук useEffect

useEffect - это функция которая принимает в себя callback в качестве первого аргумента. Этот callback будет запускаться при каждой отрисовки JSX. Вторым аргументом useEffect принимает массив зависимостей, т.е. элементами массива будет значение из локального state или из state который пришел через пропсы. При каждой отрисовки useState будет сравнивать предыдущее состояния второго параметра с текущим (пришедшим вновь), если изменений нет, то callback запускаться не будет

Пример

Если второй аргумент не передан, то callBack функция переданная первым аргументом будет запускаться при каждом рендере компоненты. Если передать пустой массив, то вызов callBack будет выполнен только один раз при монтировании компоненты, это равносильно методу жизненного цикла componentDidMount
