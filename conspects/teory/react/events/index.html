События в React

Для того что бы обработать событие в React, мы просто добавляем атрибут тегу on и далее с Большой буквы пишем название события. Далее в качестве значения мы указываем JS код (в фигурных скобках)

Но писать в атрибуте JS код таким образом считается плохой практикой. Поэтому лучше за пределами JSX кода объявить функцию и уже тегу в качестве значения атрибута передать ее
