jquery в WordPress

Может случиться так что в Wordpress не будут работать скрипты которые используют библиотеку jquery. Дело в том что WordPress подключает свою библиотеку jquery, в которой не предусмотрено обращаться к jquery с помощью символа "$". Что бы jquery работал корректно, необходимо либо обращаться к jquery через константу "jQuery" (соблюдая регистр) либо в обернуть весь наш код в такую обертку:

jQuery(document).ready(function($){ });

При этом в файле functions.php прописывать добавление библиотеки НАДО!
