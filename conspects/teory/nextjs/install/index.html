Установка среды

Обычная установка

npx create-next-app@latest

Для разваорачивания проекта с TipeScript вводим команду

npx create-next-app@latest --typescript

Вводим именно эту команду, после этого мастер установки запросит название будущего проекта

После установки что бы запустисть среду разработки вводим в консоле:

npm run dev

В корне приложения есть папка pages - это папка роутинга. В ней есть файл index.js. Дефолтная компонента этого файла будет отрисовываться если пройти по корневому url на нашем сайте. Если мы создадим например файл about.js, то дефольтный компонент, будет доступен по url localhost:3000/about

В Next JS в компонентах не нужно прописывать в импортах React, он будет доступен и так.
