Создание приложения

Устновка Meteor

Для начала необходимо глобально установить пакеты Meteor

Linux / Mac

curl https://install.meteor.com/ | sh

Windows

npm install -g meteor

Создаем Meteor проект

meteor create simple-todos-react

simple-todos-react - название нашего проекта

Meteor создаст все необходимые файлы. Файлы, расположенные в каталоге client, настраивают клиентскую сторону (веб), например, client / main.jsx, где Meteor отображает основной компонент вашего приложения в HTML.

Кроме того, в каталоге сервера, в котором Meteor настраивает серверную часть (Node.js), можно увидеть, что server / main.js инициализирует вашу базу данных MongoDB некоторыми данными. Нам не нужно устанавливать MongoDB, поскольку Meteor предоставляет встроенную версию, готовую для использования.

Что бы запустить приложение в консоле вводим команду:

meteor run

C этого момента Meteor будет синхронизировать ваше приложение со всеми вашими изменениями.

код React будет находиться внутри каталога import / ui, а файл App.jsx является корневым компонентом вашего приложения React To-do.
