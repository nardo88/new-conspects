Создание схемы

Наша БД будет иметь следующую структуру

Здесь у нас две таблицы. Person - это пользователь, имеет следующие ключи:

  • id - уникальный идентификатор SERIAL
  • name - имя пользователя строковое значение VARCHAR
  • surname - фамилия пользователя строковое значение VARCHAR

Вторая таблица имеет поля:

  • id - уникальный идентификатор SERIAL
  • title - заголовок статьи строковое значение VARCHAR
  • content - содержимое статьи строковое значение VARCHAR
  • user_id - id пользователя внешний ключ FK
