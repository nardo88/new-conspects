Работа с файлами

Для работы с фалами html форме добавляется атрибут enctype со значением - multipart/form-data. Для выбора файла используется input c типом file

Информация об отправленных файлах хранится в глобальной переменной $_FILES

$_FILES - это объект у которого будут ключи названные так же как name у тега input/file. Этот ключ будет в себе содержать ассоциативный массив который будет выглядеть вот так, если его вывести xxthtp var_dump, т.е. он будет содержать всю информацию о файле
