Создание проекта на MongoDB

Заходим на сайт mongoDb авторизовываемся и создаем новый проект:

Указываем имя проекта, жмем next после чего жмем Create Project и наконец жмем Build a DataBase. Откроется окно где нужно выбрать тариф, выбираем free

На следующем этапе выбираем расположение и жмем Create Cluster

Создание кластера может занять некоторое время

Что бы не ждать просто так указываем логин и пароль пользователя который будет иметь доступ к БД

Жмем create user. Теперь нам надо настроить IP адреса, делается это на вкладке Network Access. Жмем Add IP adres

В открывшемся окне жмем Allow Access from Anywhere (при необходимости можно настроить доступ) Жмем confirm

После того как кластер создан переходим на вкладку DATABASES и жмем кнопку Connect

В открывшемся окне выбираем Connect your application и копируем URL до нашего кластера

Подключение

В нашем проекте создаем переменную куда сохраняем ссылку до кластера

Что бы работать с БД нам необходимо установить пакет mongoose

npm i mongoose

Импортируем mongoose

Далее создадим ассинхронную функцию
