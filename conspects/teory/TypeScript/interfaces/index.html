Интерфейсы

Интерфейсы - по сути мы создаем некоторый тип, который служит для объектов или классов где мы указываем какие поля, какие функции и какие вообще элементы должны присутсоввать у этих объектов.

Для того что бы завести interface мы используем ключевое слово interface

  • Rect - имя интерфейса
  • readonly - свойство доступно только для чтения
  • color? - знак вопроса говорит что свойство color не обязательное

Использование интерфейсов у классов

Создадим интерфейс

Теперь создадим класс на основании этого интерфейса

  • implements - говорит о том что класс наследуется от интерфейса
  • date: Date - входной параметр имеет тип Date
  • void - означает что метод ничего не возвращает
