Перевод NextJS на TypeScript

Вообще изначально создавая проект NextJS можно указать что он будет написан на TS

npx create next-app app-name --typescript

Если же проект изначально не был на TS то мы сначала устанавливаем пакеты TS

npm install --save-dev typescript @types/react @types/node

После этого переименовываем файлы из JSX в TSX, из JS в TS и работаем
