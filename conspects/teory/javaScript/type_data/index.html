Типы данных

Есть 8 основных типов данных

  • number - число
  • string - строка
  • boolean - логический тип
  • null - ничего не содержит
  • object - объект
  • function - функция

Оператор typeof

Этот оператор возвращает тип аргумента. У typeof есть два синтаксиса:

как оператор - typeof x

как функция - typeof(x)
