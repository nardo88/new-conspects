Многострочные стрелочные функции

Arguments

Если число аргументов в вызове функции превышает число имен параметров, функция лишается возможности напрямую обращаться к неименованным значениям. arguments - это объект, подобный массиву, позволяющий извлекать переданные функции значения по их номерам, а не по имени.

arguments - имеет свойство length, которое отображает количество введенных аргументов

Важно Arguments не доступен у стрелочных функций

Косвенный вызов (call, apply, bind)

Что бы вызвать фнкцию fun (без аргументов) как метод объекта obj, можно воспользоваться любым из методов call, bind или apply

Анонимная функция

Анонимной называют функцию которая не имеет имени после ключевого слова function

Анонимные фкнции могут вызывать сами себя, для этого их оборачивают в круглые скобки и в конце добавляют еще одни что бы вызвать

Чистая функция

Чистая функция - это такая функция которая принимает данные и возвращает результат операции с этими данными. Чистая функция не может изменять входные данные и она не выводит ничего в консоль. Чистая функция использует только входные параметры. она не обращается к глобальным переменным

Свойства функции

По типу данных функция является объектом? хотя и typeof вернет 'function'. У функции можно вызвать свойство length которое вернет количество ожидаемых аргументов
