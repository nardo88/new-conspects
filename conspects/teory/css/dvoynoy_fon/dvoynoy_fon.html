Установка двойного фона

Для того что бы установить двойной фон, необходимо в свойстве background-image указать два атрибута url через запятую

Дополнительные свойства, таие как:

  • background-repeat
  • background-position
  • background-size
так же могут принимать по несколько значений через запятую, для первого и последующих фоновых изображений соответственно.

То изображение, которое будет указано первым оно будет находится выше остальных.
