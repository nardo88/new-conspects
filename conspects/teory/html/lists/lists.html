Теги создания списков

  1. Пронумерованный список

синтаксис

<ol>

<li>Текст 1</li>

<li>Текст 1</li>

</ol>

результат

  1. Текст 1
  2. Текст 1

атрибуты:

type – тип нумерации (1, а, А, I, i)

start – с какого значения начнется нумерация

reversed – значение этого атрибута – «reversed», нумерация будет в обратном порядке



  1. Не пронумерованный список

синтаксис

<ul>

<li>Текст 1</li>

<li>Текст 2</li>

</ul>



результат

  • Текст 1
  • Текст 2



атрибуты:

type – тип (disk, circle, square)





list-style-type: none; - отключит отображение маркерев списка
