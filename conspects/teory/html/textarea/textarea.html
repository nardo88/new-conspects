Тег <textarea>…</textarea>

Поле <textarea> представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста.



атрибуты

cols - Ширина поля в символах.

Disabled - Блокирует доступ и изменение элемента.

Form - Связывает текстовое поле с формой по её идентификатору.

Maxlength - Максимальное число введенных символов.

Name - Имя поля, предназначено для того, чтобы обработчик формы мог его идентифицировать.

Placeholder – отображает текст на заднем фоне, который исчезнет после начала ввода текста

Rows - Высота поля в строках текста.

Стилизация тега:

  • resize: none; - запретить изменение размера области ввода
  • overflow: auto; - в браузере IE полоса прокрутки отображается всегда, ставим это свойство что бы полоса появлялась только при необходимости
