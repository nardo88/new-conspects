Ссылки

Синтаксис

<a href="адрес ссылки"> текст ссылки </a>

атрибуты:

target – у этого атрибута есть две функции

- первая позволяет нам определить способ открытия ссылки. Можно отрывать ссылку как в новом окне браузера, так и на текущей странице. Для этого надо указать значение target либо "_blank" (откроет в новом окне) либо "_self" (в текущем окне)

- вторая функция позволяет привязать ссылку к фрейму. Для этого нам понадобится имя фрейма.

<a href="адрес ссылки" target="oblast"> Текст ссылки</a>

<iframe src="ссылка для фрейма" name="oblast" ></iframe>

В первой строке в значении атрибута target мы в кавычках указали имя объекта iframe. Теперь при нажатии на ссылку содержимое фрейма заменится html документом, который указан в ссылке.

title – всплывающая подсказка (пример: <a href= "адрес" title= "подсказка">ссылка</a> )



Переходы по разделам страницы

Ссылаться можно не только на другие html документы, но и на разделы текущей страницы. Для этого используется атрибут «name».

Синтаксис

<a href= "#текст">Текст ссылки </a>

<a name= "#текст"></a>

Первая строка это тег ссылки, а вторая является маяком куда надо будет переместиться после нажатия ссылки.

Пример

<a href= "#example">ссылка </a>

<a name= "#example "></a>




Ссылка на картинку

<a href="picture.jpg">Сылка на картинку</a> - при таком варианте картинка откроется в текушем окне. Если в ссылку добавить атрибут download - то при нажатии на ссылку картинка начнется скачиваться

Пример:

<a href="pictures.jpg" download="">Скачать картинку</a>
