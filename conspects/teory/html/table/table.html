Создание таблиц

синтаксис

<table>

<tr>

<td></td> <td></td>

<tr>

</table>

комментарии

tr – строка

td – ячейка в строке

атрибуты

colspan – с помощью этого атрибута можно объединять ячейки.

пример: <td colspan = "3"> </td> - т.е. объединятся три ячейки

width – ширина таблицы. Можно указать в процентах, в этом случае таблица будет изменяться вместе с окном браузера.

пример: <table width = "100%">

align - Задает выравнивание таблицы по краю окна браузера. Допустимые значения: left — выравнивание таблицы по левому краю, center — по центру и right — по правому краю. Когда используются значения left и right, текст начинает обтекать таблицу сбоку и снизу.

Bgcolor - Устанавливает цвет фона таблицы.
