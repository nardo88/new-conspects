Атрибуты

  • autoplay - видео начнет проигрываться сразу после загрузки страницы
  • сontrols - появится стандартная панель управления видео
  • loop - видео будет зациклено
  • preload - Используется для загрузки видео вместе с загрузкой веб-страницы. Этот атрибут игнорируется, если установлен autoplay. (значение по умолчанию - none)
    • none - Не загружать видео.
    • metadata - Загрузить только служебную информацию (размеры видео, первый кадр, продолжительность и др.).
    • auto - Загрузить видео целиком при загрузке страницы.
  • poster - Указывает адрес картинки, которая будет отображаться, пока видео не доступно или не вопроизводится. (poster="./img1.jpg")
  • src - Указывает путь к воспроизводимому видеоролику.
  • muted - если указать значение "muted", то у видео будет отключен звук
  • playsinline - атрибут для корректного отображения видео в safari
<video class="video-girl__item" muted="muted" playsinline="" autoplay="" loop="" preload="auto"> <source src="./video/promo.mp4" type="video/mp4"><!-- MP4 для Safari, IE9, iPhone, iPad, Android, и Windows Phone 7 --> <source src="./video/promo.webm" type="video/webm"><!-- WebM/VP8 для Firefox4, Opera, и Chrome --> </video>

Часто встают проблемы с браузером safari. Видео не проигрывается. Что делать? Во-первых, видео должно быть в формате mp4. Во-вторых, должны стоять атрибуты muted и playsinline. И в-третьих, на устройстве (iphone, mac) должен быть отключен энергосберегающий режим.
