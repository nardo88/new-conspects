Плагины

JS Quick Console - позволяет быстро призвать console.log (Ctrl + Shift + L)

Live Server - крутой аналог Browser Sync

Code Runner - Позволяет запускать код JS в консоли отладки

open in browser - позволяет открывать из редатора файлы в браузере

All Autocomplete - позволяет автозаполнять классы или переменные которые мы уже объявили

Autoprefixer - автоматом проставляет префиксы

Beautify - приводит код в красивое положение

ESLint - проверяет правильность кода JS

Sass - Для написания Sass (Scss) файлов

Polacode-2020 - делает скриншоты кода. Для этого зажимает Ctrl + Shift + P - вводим >polacode и жмем Enter

Color Picker - позволяет вызвать палитру цветов. Длы высова просто наводим курсор на цвет (#000)

Color Highlight - подсвечивает цвета в CSS

AutoFileName - позволяет указывать автоматом имена в свойстве src для вызова ставим ./

Bracket Pair Colorizer - этот плагин подсвечивает скобки в JS

Highlight Matching Tag - подсвечивает теги в html

Auto import - Автоматически вставляет import при использовании компоненты в react

eCSStractor for VSCode - позволяет копировать разметку по БЭМ в SCSS (сочетание клавиш - shift + f1)

Live SASS Compiler - Этот плагин генерирует css файлы (можно использовать вместо сборщика Preproc)

Remote - SSH - Для подключения к удаленному репозиторию
