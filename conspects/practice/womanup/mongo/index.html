Работа с Mongo DB через shell

  • meteor mongo - подключение к mongo shell внутри среды разработки
  • db.users.find().pretty() - получить данные из коллекции (user) в удобочитаемом виде
  • db.settings.insert({title: 'ModeratorUserId', name: 'ModeratorUserId', value: 'EkrNtd6wH'}) - добавляем в настройки учетную запис методиста
  • db.users.update({_id: 'EkrNtd6wH'}, {$set: {role: 'admin'}}) - изменение роли пользователя
