    #checkbox
  • display: none; - прячем наш checkbo т.к. квадратик будем рисовать сами.

    .label
  • padding-left: 20px; - сделаем небольшой отступ для будущего квадратика

    .label::before - рисуем квадратик с помощью псевдоэлемента before
  • content: ''; - пустой контент
  • display: inline-block; - делаем элемент блочным что бы задавать высоту и ширину
  • width: 18px; - ширина
  • height: 18px; - высота
  • border: 2px solid #061e37; - создаем обводку квадратика
  • border-radius: 3px; - делаем его немного закругленным
  • vertical-align: middle; - выравниваем текст по вертикали относительно квадратика
  • margin-left: -10px; - двигаем квадратик по оси X
  • margin-right: 10px; - двигаем квадратик по оси X

    input[type="checkbox"]:checked + .label::before - если input с типом "checkbox" в состоянии checked (т.е. нажат) то применяются стили к следующему ( + ) элементу ( .label::before ). Т.е. мы задаем стили как будет выглядеть квадратик после того как мы нажали по нему (или по тексту). Можно делать что угодно, например установить background-image: и использовать любую картинку.
  • content: '';
  • text-align: center; - выравниваем содержимое квадратика по центру
  • font-size: 15px; - размер содержимого квадратика
  • background-image: url(checked-symbol.png); - установили картинку (иконка галочки)
  • background-size: cover; - подогнали размер картинки.
