Преобразование данных из БД (MySQL) d JSON

Ссылка на GitHub

Алгоритм действий следующий

  1. Получить данные из БД
  2. Создать пустой массив
  3. Создать класс который будет создавать объект с ключами
  4. Пробежаться по массиву полученному из БД и на каждой итерации создавать объект на основании класса и запушить этот объект в новый массив
  5. После этого новый массив с объектами кодируем в JSON и возвращаем его через echo
