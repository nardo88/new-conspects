Код CSS

Заострю внимание только на сточках 19 и 20. Эти свойства применяются к родительскому элементу карточек

  • perspective: 1000px; - CSS свойство perspective определяет на сколько элемент удален от точки обзора. Свойство определяет перспективу обзора элемента, создавая для пользователя иллюзию глубины. Чем больше значение, указанное для перспективы, тем дальше от пользователя расположен элемент.
  • transform-style - Определяет, как дочерние элементы должны отображаться в 3D-пространстве. Это свойство должно использоваться совместно с transform.

Код Java Script

Первое что мы делаем - это обьявляем переменную cards в которую помещаем все элементы с классом card. Это родительские элементы наших карточек.

Затем мы запускаем цикл. Для чего цикл? для того что бы мы могли обращаться к каждому элементу в отдельности.

Цикл читается так начиная с 0 и до количства элементов, каждому элементу мы применяем функцию addEventListener которая будет прослушивать событие mousemove - наведение курсора мыши и mouseout - когда курсор убран с элемента. В каждое событие будет выполняться своя функция. Рассмотрим каждую отдельно.

Функция rotateStart - имеет входящим параметром событие event. Это событие фиксирует все что только можно. В теле функции мы объявляем переменную cardItem и в нее мы помещаем уже дочерний элемент (саму карточку). Теперь обращаясь к этой переменной мы можем задавать ей стили, а именно будем регулировать значение такого свойства CSS как transform (это свойство отвечает за перемещение или вращение элемента). В это свойство мы помещаем значение rotate (именно это значение отвечает за вращение элемента по осям Х и Y). Теперь надо указать на сколько deg (т.е. градусов) надо вращать элемент.

А это значение мы будем брать с помощью события event. У события event есть метод получения координа курсора по осям X и Y. Это методы offsetY и offsetX. к этим методам применяем математические выражения что бы вращеные было корректным и все, анимация получилась.
